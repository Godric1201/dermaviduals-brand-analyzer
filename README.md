# Dermaviduals Brand Analyzer

AI-powered visibility analysis tool for skincare brands.

## Features
- Prompt-based AI recommendation testing
- Brand visibility scoring
- Competitor benchmarking
- Strategy & content generation
- Streamlit dashboard

## Tech Stack
- Python
- Streamlit
- OpenAI API
- Pandas

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
